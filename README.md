# ✅ Gerenciador de Tarefas (Task Manager)

https://gerenciador-tarefas-reac.vercel.app/

## 💻 Sobre o Projeto
Este é um aplicativo de lista de tarefas (To-Do List) moderno desenvolvido com React. O projeto foi construído como forma de estudo para consolidar conceitos fundamentais do desenvolvimento Front-end, como gerenciamento de estados (Hooks), consumo de APIs externas e persistência de dados no navegador.

## ✨ Funcionalidades
- **Criar tarefas:** Adicione novas tarefas com título e descrição.
- **Gerenciar status:** Marque tarefas como concluídas (riscadas) ou pendentes.
- **Excluir:** Remova tarefas que não são mais necessárias.
- **Detalhes da Tarefa:** Navegue para uma página exclusiva para ler os detalhes de cada tarefa.
- **Persistência de Dados (LocalStorage):** As tarefas não somem ao dar F5! O aplicativo salva suas informações diretamente no navegador.
- **Consumo de API:** Na primeira carga do site (se o navegador estiver vazio), o aplicativo consome a API pública JSONPlaceholder para popular a lista com tarefas de exemplo.

## 🚀 Tecnologias Utilizadas
- **[React](https://reactjs.org/)** (com **Vite** para build super rápido)
- **[Tailwind CSS](https://tailwindcss.com/)** para estilização utilitária e responsiva
- **[React Router DOM](https://reactrouter.com/)** para navegação e rotas dinâmicas
- **[Lucide React](https://lucide.dev/)** para a biblioteca de ícones
- **[UUID](https://www.npmjs.com/package/uuid)** para geração de identificadores únicos e seguros
